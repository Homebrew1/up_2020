The svrx plugin for svrx-my-plugin

## Usage

> Please make sure that you have installed [svrx](https://svrx.io/) already.

### Via CLI

```bash
svrx -p svrx-my-plugin
```

### Via API

```js
const svrx = require('@svrx/svrx');

svrx({ plugins: [ 'svrx-my-plugin' ] }).start();
```
