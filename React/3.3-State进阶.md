}

### 05. 有状态组件 & 无状态组件

**stateful 和 stateless 的区别**

+ stateful
  + 类组件
  + 有状态组件
  + 容器组件
+ stateless
  + 函数组件
  + 无状态组件
  + 展示组件

**尽可能通过状态提升原则，将需要的状态提取到父组件中，而其他的组件使用无状态组件编写**
